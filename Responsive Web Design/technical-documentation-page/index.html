Introduction

Python is easy to learn, highly readable, and simple to use. It has a clean and english-like syntax which requires less coding and let the programmer focus on the business logic rather than thinking about the nitty-gritty of the language.

One of the tools used by Python developers is the Python Interpreter. It operates somewhat like the Unix shell: when called with standard input connected to a tty device, it reads and executes commands interactively; when called with a file name argument or with a file as standard input, it reads and executes a script from that file. In Linux, you need to open a terminal and callit like this:

python3 [someSript]
Hello World

In your favourite text editor, create a new file, name it hello_world.py, and write the next code:

print("Hello World")

Then call the Python 3 Interpreter with this file to see the words "Hello World" printed on the console

Variables

In Python, when you need to store some information, you can use variables by giving a name to some chunk of information. For example, to create a variable called foo that stores the value 5, you would need to write something like this:

foo = 5

Suppose, now, that you want to add one to this variable and print the result of this operation. Then you would need something like this:

print(foo + 1)

Python is dinamically typed, like javascript, this means that you don't have to declare what type each variable is.

Lists and dictionaries

Loops

When you need to repeat the execution of some lines of code a certain amount of times, Python gives us the option to use loops.

The are two primitive loops commands: for and while

  • If you know how many times you need to repeat some action, say ten times, you could use:
    for x in range(0,10):
      print(x)
    This will print the first ten numbers in order.
  • When you dont know the exact times you need to repeat it but you have a condition that must be met to stop, you can use a while loop:
    x = 0
    while x*x < 1000:
      print(x)
      x = x + 1
    This will print the all the numbers whose square is less than one thousand. Note that, in this case, we need to manually update the variable x.
  • The last form of loops allows us to apply some code to each item of a collection, like a list.:
    days = ["Monday", "Tuesday", "Wednesday", "Thursday", "Friday", "Saturday", "Sunday"]
    for day in days:
      print(day)
    This code will print all the strings in the list days
